📝 To-Do List (CLI Project)

A simple, interactive To-Do List application built using Python.
This project focuses on the basics of programming while solving a real-life problem — **managing tasks efficiently**.

🌱 Why I Built This

I wanted to create something practical while learning Python fundamentals like:
* Functions
* Lists & Dictionaries
* Loops and Conditions
Instead of just solving textbook problems, this project helped me understand how real programs are structured.

⚙️ What This Program Can Do

* Add tasks with subject and time
* View all tasks in a clean format
* Mark tasks as completed
* Assign priority (High 🔴 / Medium 🟡 / Low 🟢)

🧠 How It Works
Each task is stored as a dictionary containing:
* Subject
* Time
* Status (pending/completed)
* Priority
All tasks are stored in a list and managed through functions.

▶️ How to Run It
1. Make sure Python is installed
2. Download or clone the repository:

git clone https://github.com/your-username/repo-name.git
3. Open the folder:
cd repo-name
4. Run the program:
python main.py

💬 Example Flow
1. Add Task
2. View Tasks
3. Completed Task
4. Exit

Simple, fast, and easy to use.

⚠️ Current Limitations
* Tasks are not saved after closing the program
* Limited error handling
* UI is basic (command line only)
