# 📝 To-Do List
A simple command-line based To-Do List application built using Python.  
This project helps users manage daily tasks by adding, viewing, and marking them as completed.

## 🚀 Features
- ➕ Add new tasks with subject and time
- 📋 View all tasks
- ✅ Mark tasks as completed
- 🎯 Assign priority (High / Medium / Low)
- 🖥️ Simple and beginner-friendly CLI interface

## 🛠️ Technologies Used
- Python (Basic concepts)
- Lists & Dictionaries
- Functions
- Loops & Conditionals
